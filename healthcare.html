Healthcare Data Science Projects

Automated Epi Report Tool (CDC Response)
Developed during an emergency response with the CDC, this tool automated daily outbreak reports across refugee camps. It parsed medical notes to identify infectious disease conditions and produced formatted PDF reports, replacing a manual process that previously required several hours. The tool was later scaled for national use in settings with limited internet access.

Healthify: Flu Surveillance & Modeling Dashboard
An interactive web application built with Python and R, featuring dynamic visualizations, forecasting tools, and epidemiological models powered by CDC FluSurv-NET data. Combines machine learning and public health analytics to support real-time disease surveillance and decision-making.
